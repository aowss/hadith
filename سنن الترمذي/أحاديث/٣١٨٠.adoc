
= الحديث رقم ٣١٨٠

[quote.hadith]
حَدَّثَنَا سُوَيْدُ بْنُ نَصْرٍ، حَدَّثَنَا عَبْدُ اللَّهِ، عَنْ يُونُسَ بْنِ يَزِيدَ، بِهَذَا الإِسْنَادِ نَحْوَهُ ‏.‏ قَالَ أَبُو عِيسَى وَأَبُو عَلِيِّ بْنُ يَزِيدَ هُوَ أَخُو يُونُسَ بْنِ يَزِيدَ وَهَذَا حَدِيثٌ حَسَنٌ غَرِيبٌ ‏.‏ قَالَ مُحَمَّدٌ تَفَرَّدَ ابْنُ الْمُبَارَكِ بِهَذَا الْحَدِيثِ عَنْ يُونُسَ بْنِ يَزِيدَ وَهَكَذَا قَرَأَ أَبُو عُبَيْدٍ ‏5.45-45(‏ وَالْعَيْنُ بِالْعَيْنِ ‏)‏ اتِّبَاعًا لِهَذَا الْحَدِيثِ ‏.‏
